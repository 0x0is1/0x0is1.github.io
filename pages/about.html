Introduction

Century-pedia: Power to the User, Powered by the User

A simple website made HTML, CSS and Javascript to proper theming features, and API consumption to provide Century-pedia of almost everything.

This Project contains data Rated by the User.

Although as of now this project is mainly read-only, but in near future, It will also accept ratings from the user.

Features

  • Template embedding based

    There are minimal amount of direct HTML has been used in the project.
    Most of them are in string form in javascript files.
    This is to avoid lagging and increasing reusabilty of loaded page.

  • Optimized with Redis

    API used in the project to provide data is completely optimized with Redis
    to decrease pmemory consumption.

  • Vanilla.js for css effects

    Even though it has not been taught in class, This project still contains
    Vanilla.js for special effects like Pagination bar, Glassmorphism effect, Neon naviagation bar effect etc.

  • REST API Based

    This project is using Ranker API to get TOP 100
    data to use in. This API consumption is with the proper acknowledgement of
    API owners, so not illegal. Although API wrapper can be added to divert reverse tracking.

    It also uses github API and other 3rd party APIs to get users and Repository data.

  • Themed

    Theme-ing has been applied in the pages with provide proper customisiation as per users need.

    Themes avaibale are:

    1. Cyan
    2. Yellow
    3. Red
    4. Green
    5. Orange
